# Airbnb Listings & Reviews Analysis Dashboard

## Project Overview

This project is an interactive Power BI dashboard built using the Maven Analytics Airbnb Listings & Reviews dataset. It analyzes Airbnb performance across 10 global cities by exploring listings, pricing, customer ratings, reviews, host verification, and seasonal demand patterns.

---

## Business Problem

Airbnb operates across multiple cities with different pricing strategies, customer preferences, and host behaviors. This dashboard helps stakeholders identify market trends, understand customer satisfaction, compare city performance, and uncover opportunities for business growth.

---

## Questions Answered

### Executive Overview

- How large is Airbnb's presence across major global cities?
- Which cities contribute the most listings?
- How has Airbnb listing growth changed over time?
- Which room types dominate the platform?

### Market & Pricing

- Which cities have the largest Airbnb markets?
- Which room types are most expensive?
- How do pricing patterns vary across room types?

### Ratings & Customer Satisfaction

- Which cities provide the best guest experience?
- Which rating metrics perform well or poorly?
- How does customer satisfaction compare across cities?

### Reviews & Customer Engagement

- Which room types receive the highest customer engagement?
- How does review activity vary throughout the year?
- Which seasons generate the most reviews?

### Host Trust & Verification

- What percentage of hosts are verified?
- How common are anonymous or unverified hosts?
- Does Airbnb provide sufficient trust signals to customers?

## Dataset

**Source**

https://mavenanalytics.io/data-playground/airbnb-listings-reviews

### Dataset Size

- 279,712 Listings
- 5.37 Million Reviews
- 182,024 Hosts
- 10 Cities
- 144 Property Types

---

## Data Model

- Listings Table
- Reviews Table
- One-to-Many Relationship using `listing_id`

---

## Tools Used

- Power BI Desktop
- Power Query
- DAX
- Data Modeling
- Power BI Visualizations

---

## Key Insights

- Paris has the highest number of Airbnb listings.
- Entire Place listings generate the majority of customer engagement.
- Hotel Rooms have the highest average nightly price.
- Mexico City records the strongest overall guest ratings.
- Review activity peaks during the summer travel season.
- More than two-thirds of Airbnb hosts are fully verified.
